# 把真实代码改成这套设计系统的样子

你的 `online-mall` 代码库其实已经做了大部分工作——它是"两层结构":
旧的紫色样式在前,后面又追加了"统一为商城风格"的干净覆盖块(CSS 后者
生效)。所以**签到日历、用户/商家工作台、商品卡、订单/库存页基本已经
是干净蓝白风格了**,前面那一堆紫色多是被盖掉的死代码。

**还没被覆盖、仍残留视觉问题的地方**主要三类:
1. `App.vue` 顶部头部:**紫渐变 + 彩虹动画**(最显眼)
2. `user/main.vue`、`favorite.vue` 等里的**玻璃白卡 / 紫色顶条**
3. 各页面**不一致的红蓝**(购物车 `#1989fa`/`#ff4d4f`、收藏页价格红)

---

## 推荐做法:一个文件搞定全站(最快)

把 **`mall-clean.css`** 复制到 `src/assets/mall-clean.css`,然后在
`src/main.js` 里**最后**引入一行(放在 Element Plus 样式之后):

```js
import 'element-plus/dist/index.css'
import '@/assets/mall-clean.css'   // ← 加这一行,全站对齐
```

这一个文件做了 **7 件事**:
1. 去掉全屏彩虹动画背景
2. 顶部头部紫渐变 → 干净白底
3. Logo / 导航链接玻璃白 → 朴素文字 + 蓝图标
4. 收藏页、用户首页的玻璃卡 / 紫色顶条 → 标准白卡
5. 商品卡统一 8px 圆角 + 灰描边 + 柔和阴影 + hover 升起
6. 价格统一为价格红 `#e4232b`(覆盖 `#ff4d4f` 等)
7. Element Plus 主色对齐商城蓝(`el-button`、单选按钮选中态等自动生效)

改完后,对照本项目两个 UI Kit 的成品核对外观即可:
- `ui_kits/web-app/index.html` (商城前台)
- `ui_kits/merchant/index.html` (商家后台)

---

## 备用做法:更克制的两文件方案

如果你只想动顶部头部、不想用全站覆盖,可以改用本文件夹里的两个小文件:

- **`app-header-clean.css`** —— 整段粘贴到 `App.vue` 的 `<style>` 末尾
  (它不是 `scoped`,追加在最后即可压制紫色)。
- **`element-theme.css`** —— 放到 `src/assets/`,在 `main.js` 里引入,
  让 Element Plus 主色变商城蓝。

两个方案**不要同时用**——选 `mall-clean.css` 就够了。

## 收尾零散项(改完上面再做)

全项目搜以下关键字,逐个替换/清理:

| 搜 | 替换为 | 说明 |
|----|--------|------|
| `#667eea` / `#764ba2` | `var(--mall-primary)` 或删除 | 残留紫色 |
| `rgba(102, 126, 234, …)` | `var(--mall-accent-bg)` 等 | 紫色透明态 |
| favorite.vue 里的 `⭐` | `<el-icon><StarFilled/></el-icon>` | 别用 emoji,用 Element 图标 |
| 写死的 `#1989fa` `#ff4d4f` | 主蓝/价格红令牌 | 让颜色单一来源 |

清理完后,可以再把 App.vue / 各 view 里早期的紫色规则段整段删除,
让源文件本身也变干净。
